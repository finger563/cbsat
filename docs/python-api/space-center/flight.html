Flight¶

class Flight¶

Used to get flight telemetry for a vessel, by calling Vessel.flight(). All of the information returned by this class is given in the reference frame passed to that method.

Note

To get orbital information, such as the apoapsis or inclination, see Orbit.

g_force¶

Gets the current G force acting on the vessel in \(m/s^2\).

Return type:float
mean_altitude¶

Gets the altitude above sea level, in meters.

Return type:double
surface_altitude¶

Gets the altitude above the surface of the body or sea level (whichever is closer), in meters.

Return type:double
bedrock_altitude¶

Gets the altitude above the surface of the body, in meters. When over water, this is the altitude above the sea floor.

Return type:double
elevation¶

Gets the elevation of the terrain under the vessel, in meters. This is the height of the terrain above sea level, and is negative when the vessel is over the sea.

Return type:double
latitude¶

Gets the latitude of the vessel for the body being orbited, in degrees.

Return type:double
longitude¶

Gets the longitude of the vessel for the body being orbited, in degrees.

Return type:double
velocity¶

Gets the velocity vector of the vessel. The magnitude of the vector is the speed of the vessel in meters per second. The direction of the vector is the direction of the vessels motion.

Return type:Vector3
speed¶

Gets the speed of the vessel in meters per second.

Return type:double
horizontal_speed¶

Gets the horizontal speed of the vessel in meters per second.

Return type:double
vertical_speed¶

Gets the vertical speed of the vessel in meters per second.

Return type:double
center_of_mass¶

Gets the position of the center of mass of the vessel.

Return type:Vector3
rotation¶

Gets the rotation of the vessel.

Return type:Quaternion
direction¶

Gets the direction vector that the vessel is pointing in.

Return type:Vector3
pitch¶

Gets the pitch angle of the vessel relative to the horizon, in degrees. A value between -90° and +90°.

Return type:float
heading¶

Gets the heading angle of the vessel relative to north, in degrees. A value between 0° and 360°.

Return type:float
roll¶

Gets the roll angle of the vessel relative to the horizon, in degrees. A value between -180° and +180°.

Return type:float
prograde¶

Gets the unit direction vector pointing in the prograde direction.

Return type:Vector3
retrograde¶

Gets the unit direction vector pointing in the retrograde direction.

Return type:Vector3
normal¶

Gets a unit direction vector pointing in the normal direction.

Return type:Vector3
anti_normal¶

Gets a unit direction vector pointing in the anti-normal direction.

Return type:Vector3
radial¶

Gets a unit direction vector pointing in the radial direction.

Return type:Vector3
anti_radial¶

Gets a unit direction vector pointing in the anti-radial direction.

Return type:Vector3
atmosphere_density¶

Gets the current density of the atmosphere around the vessel, in \(kg/m^3\).

Return type:float

Note

Calculated using KSPs stock aerodynamic model, or Ferram Aerospace Research if it is installed.

dynamic_pressure¶

Gets the dynamic pressure acting on the vessel, in Pascals. This is a measure of the strength of the aerodynamic forces. It is equal to \(\frac{1}{2} . \mbox{air density} . \mbox{velocity}^2\). It is commonly denoted as \(Q\).

Return type:float

Note

Calculated using KSPs stock aerodynamic model, or Ferram Aerospace Research if it is installed.

static_pressure¶

Gets the static atmospheric pressure acting on the vessel, in Pascals.

Return type:float

Note

Calculated using KSPs stock aerodynamic model. Not available when Ferram Aerospace Research if it is installed.

aerodynamic_force¶

Gets the total aerodynamic forces acting on the vessel, as a vector pointing in the direction of the force, with its magnitude equal to the strength of the force in Newtons.

Return type:float

Note

Calculated using KSPs stock aerodynamic model. Not available when Ferram Aerospace Research if it is installed.

lift¶

Gets the aerodynamic lift currently acting on the vessel, as a vector pointing in the direction of the force, with its magnitude equal to the strength of the force in Newtons.

Return type:Vector3

Note

Calculated using KSPs stock aerodynamic model. Not available when Ferram Aerospace Research if it is installed

drag¶

Gets the aerodynamic drag currently acting on the vessel, as a vector pointing in the direction of the force, with its magnitude equal to the strength of the force in Newtons.

Return type:Vector3

Note

Calculated using KSPs stock aerodynamic model. Not available when Ferram Aerospace Research if it is installed

speed_of_sound¶

The speed of sound, in the atmosphere around the vessel, in \(m/s\).

Return type:float

Note

Not available when Ferram Aerospace Research if it is installed.

mach¶

The speed of the vessel, in multiples of the speed of sound.

Return type:float

Note

Calculated using KSPs stock aerodynamic model, or Ferram Aerospace Research if it is installed.

equivalent_air_speed¶

The equivalent air speed of the vessel, in \(m/s\).

Return type:float

Note

Not available when Ferram Aerospace Research if it is installed.

terminal_velocity¶

The current terminal velocity of the vessel, in \(m/s\). This is the speed at which the drag forces cancel out the force of gravity.

Return type:float

Note

Calculated using KSPs stock aerodynamic model, or Ferram Aerospace Research if it is installed.

angle_of_attack¶

Gets the pitch angle between the orientation of the vessel and its velocity vector, in degrees.

Return type:float
sideslip_angle¶

Gets the yaw angle between the orientation of the vessel and its velocity vector, in degrees.

Return type:float
total_air_temperature¶

The total air temperature of the atmosphere around the vessel, in Kelvin. This temperature includes the static_air_temperature and the vessel’s kinetic energy.

Return type:float
static_air_temperature¶

The static (ambient) temperature of the atmosphere around the vessel, in Kelvin.

Return type:float
stall_fraction¶

Gets the current amount of stall, between 0 and 1. A value greater than 0.005 indicates a minor stall and a value greater than 0.5 indicates a large-scale stall.

Return type:float

Note

Requires Ferram Aerospace Research

drag_coefficient¶

Gets the coefficient of drag. This is the amount of drag produced by the vessel. When calculated using Ferram Aerospace Research it depends on air speed, air density and wing area.

Return type:float

Note

Requires Ferram Aerospace Research

lift_coefficient¶

Gets the coefficient of lift. This is the amount of lift produced by the vessel, and depends on air speed, air density and wing area.

Return type:float

Note

Requires Ferram Aerospace Research

pitching_moment_coefficient¶

Gets the pitching moment coefficient.

Return type:float

Note

Requires Ferram Aerospace Research

ballistic_coefficient¶

Gets the ballistic coefficient.

Return type:float

Note

Requires Ferram Aerospace Research

thrust_specific_fuel_consumption¶

Gets the thrust specific fuel consumption for the jet engines on the vessel. This is a measure of the efficiency of the engines, with a lower value indicating a more efficient vessel. This value is the number of Newtons of fuel that are burned, per hour, to product one newton of thrust.

Return type:float

Note

Requires Ferram Aerospace Research

far_status¶

Gets current status message from Ferram Aerospace Research.

Return type:string

Note

Requires Ferram Aerospace Research
